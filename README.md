# Anomaly Detection: Stock Market
# Dataset
## Price and Volume Data for all US Stocks 
This dataset is comprised of over 14 million observations and 7 columns. The variables consist of:
- Symbol
- Date
- Open
- High
- Low
- Close
- Volume
